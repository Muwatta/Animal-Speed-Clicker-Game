    Game Guide

    Overview: This is a 60‑second game where a random animal (displayed as an emoji) is generated automatically. You must click the corresponding animal button that matches the generated animal.

    How to Play:
    1. Enter your name and click "Start Game". A loading screen will appear before the game begins.
    2. Once the game starts, the first random animal is generated automatically.
    3. Click the animal button that matches the displayed animal.
    4. After you answer, feedback is shown for 2 seconds before the next animal is generated.
    5. If you score 6 or higher, the animal buttons will dance for 10 seconds as a celebration!

    Additional Features:
    - Reset Game: Restart the game without reloading the page.
    - Leaderboard: View the top 5 highest scores saved locally.

    About This Game

    Animal Speed Clicker Game is an interactive web game built using HTML, CSS, and JavaScript. Enjoy a deep green themed, responsive experience with automatic animal generation, dynamic feedback, and a fun victory celebration!

    This project was created as a fun way to practice frontend development skills.
